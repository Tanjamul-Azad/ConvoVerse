## ğŸ§­ Project Overview

**ConvoVerse** is an HCI-focused, LLM-powered multi-agent interactive system designed to support introverted and socially hesitant learners in developing confidence for communication and participation.

Rather than forcing direct real-world interaction, ConvoVerse provides a **psychologically safe, low-pressure simulated environment** where users can practice conversations, observe social dynamics, and gradually prepare for real-life participation.

The system positions AI as a **supportive facilitator**, not a replacement for human interaction.
